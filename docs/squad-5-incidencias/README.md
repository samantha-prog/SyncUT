## Misión del módulo
Desarrollar el sistema de buzón de quejas, reportes y alertas tempranas mediante un tablero visual tipo semáforo.

## Alcance funcional
- Registro de incidencias por usuarios.
- Clasificación por prioridad y estado.
- Visualización tipo semáforo (verde, amarillo, rojo).
- Seguimiento y cierre de incidencias.
- Reportes para coordinación y administración.

## Entregables mínimos
- Formulario de captura de incidencia.
- Listado de incidencias con filtros por estado y prioridad.
- Tablero semáforo para alertas tempranas.
- Vista de detalle y trazabilidad de cambios.

## Dependencias
- Bloqueo principal: Equipo 2 (autenticación, roles y permisos).
- Integración recomendada: Equipo 4 (alertas por correo).
