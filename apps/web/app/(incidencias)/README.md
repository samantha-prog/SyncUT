## Objetivo
Construir el buzon de quejas con seguimiento y alertas tempranas en un tablero tipo semaforo.

## Funciones del modulo
- Captura de incidencias.
- Clasificacion por prioridad y estado.
- Visualizacion semaforo (verde, amarillo, rojo).
- Seguimiento y cierre de casos.
- Reportes para coordinacion.
