## Objetivo
Implementar un canal de atencion inteligente y automatizada para dudas de la comunidad estudiantil.

## Funciones del modulo
- Conversacion guiada para preguntas frecuentes.
- Respuestas automatas por categoria.
- Escalamiento a atencion humana.
- Registro de conversaciones y metricas.
